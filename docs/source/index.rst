Welcome to go-cerberus documentation!
=====================================

**go-cerberus** is a Go client library for communicating with the cerberus (by a11n.io)
backend, whether you're using the cloud hosted backend, or have installed in on-premises in your own datacenter.

Check out the :doc:`usage` section for further information, including
:ref:`getting started` on the project.

.. note::

   This project is under active development.
